# Cactus–Molasses Replacement Dataset

This repository contains the Excel dataset and analysis files for a study evaluating cactus (Opuntia ficus-indica) as a partial or full replacement for molasses in dairy cow diets.

## Files
- cactus_feed_data.xlsx: Raw experimental dataset
- analysis_code.R: Statistical analysis (R software)

## Key Variables
- Treatment (T1, T2, T3, T4)
- Feed intake (kg/day)
- Milk yield (L/day)
- Milk composition (% fat, protein, lactose, SNF)
- Economic parameters (cost, return, BCR)

## Description
The study investigates the effects of cactus inclusion on feed intake, milk production, milk quality, and feed efficiency under dryland conditions.
